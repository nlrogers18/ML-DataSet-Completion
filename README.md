# ML-DataSet-Completion

- Given Training Data, Labels and Test Data pick 5 classification models, Train and Cross-Validate
- Gene expressions often contain missing values; estimate the microarray values as accurately as possible
## Root Directory

- **`Project_Report`** 
Comprehensive descripton of the project parameters and my inital thought process for approaching each dataset. Also includes the reasons for each classification model chosen for each Dataset.
- **`ML_project.ipynb/.zip`**
Main notebook file and zip for ease of acess
- **`Outputs`** 
Contains each classification .txt for each dataset classified in this project and the attempted completions for the 2 gene expression sets. 
- **`48506850Project`** 
The provided .txt files with the unclassified data/labels and the incomplete gene expression sets. 


## Requirements to Run

To run this project, you will need the following dependencies:

### Python Libraries

- **`pandas`**: For data manipulation and analysis.
- **`numpy`**: For numerical operations and array handling.
- **`scikit-learn`**: For machine learning algorithms and data preprocessing, including `train_test_split`, `RandomForestRegressor`, and `Pipeline`.
